What Changed

Key improvements in Claude 2

📖
100K Context Window

Process entire books, large codebases, or lengthy legal documents in a single pass. 100,000 tokens ≈ 75,000 words — available from day one.

🛡️
2× Safer

Dramatically fewer harmful or misleading outputs. Improved Constitutional AI training made Claude 2 approximately twice as safe as Claude 1 on red-team evals.

💻
Stronger Coding

HumanEval jumped from 56% to 71.2%. Claude 2 can write, debug, and explain code across Python, JS, Java, and more with substantially higher accuracy.

🔧
Tool Use (2.1)

Claude 2.1 introduced function/tool calling, letting developers connect Claude to external APIs, databases, and custom actions programmatically.

🎯
Better Instruction Following

Improved adherence to complex, multi-part instructions. Claude 2 stays on task longer and respects formatting, constraints, and persona requirements more reliably.

📉
Reduced Hallucination

Especially in Claude 2.1 — when uncertain, Claude 2 is more likely to acknowledge the gap rather than fabricate confident-sounding but incorrect answers.

Release History

Claude 1 → 2 timeline

March 2023
Claude 1.0 — Public launch

Anthropic's first generally available model. Available via API with a 9K token context window. Trained with Constitutional AI and RLHF. Established the helpful, harmless, honest foundation.

May 2023
Claude 1.3 — Incremental improvements

Refined version of the Claude 1 family with improved instruction following and reduced refusals on benign queries. Widely used by early API partners.

May 2023
Claude 100K — Extended context preview

A special context-extended version of Claude 1 offering 100,000 tokens for select partners — a preview of what Claude 2 would deliver at scale.

July 2023
Claude 2.0 — Major release

Launched on claude.ai and via API. 100K context, stronger coding (71.2% HumanEval), safer outputs, better at following complex instructions. Available in US and UK.

November 2023
Claude 2.1 — Tool use & accuracy

Introduced tool use (function calling), extended to 200K context for API users, reduced hallucination rate, and improved precision on long documents. System prompt support finalized.
